Bit Form Integration With Dropbox

Dropbox is a popular cloud storage platform. Now you can send form submission attachments to dropbox via bit form. The file-upload field is a great way to attach files when submitting forms. In this way, many necessary files can be collected. But if you have too many file attachments, it can have a bad effect on WordPress and can create messy storage.

  1. At first, select the form name you want to apply the Dropbox integration after that, follow the below instructions.

    Follow the path:

    Settings=> Integrations => Click on (+) => Click on ‚ÄòDropbox`

  2. Integration Name: Set the proper integration name. Default it's taken ‚ÄòDropbox Integration‚Äô.

  3. After that, To get the API key & API Secret, click on Dropbox API Console.

    1. Choose an API: Scoped access
    2. Choose the type of access you need: Full Dropbox
    3. Type Name your app and click on Create App button.
    4. Go to Permissions Tab: Permissions that allow your app to view and manage > you need to enable all scopes.
      Now copy the API key and API secret, then paste it into the integration‚Äôs dashboard.
    5. After that, to get the Access Code, click on Dropbox Access Code. And Create a Access Code. Now copy the Generated Access Code and paste it into the integration‚Äôs dashboard. And Click on ‚ÄúAuthorize‚Äù button.

  4. Click on Next to set up the other integration settings ahead.

Integration Process‚Äã

  1. After successful authorization, you will find the Map Field option. Select the folder Name from the Drop-Down list that you want to send the files to the folder.

  1. Also, if you want to delete the uploaded file from WordPress, enable the 'Delete File From WordPress' option. the Dropbox integration is ready.

  1. Now click on Next button. And save the integration by clicking the Finish & Save Button.

  2. Now, you have to set conditional logic to trigger the integration for the submitted data. When you set conditional logic, remember some points ‚Äì set Action behavior = Always & specify the integration name you have created for Dropbox. If you don‚Äôt select the integration name properly, then the integration will not work. And update the form by clicking the ‚ÄúUpdate‚Äù button.

  3. After submitting any data, you can see the record both on the Dropbox account & response page of Bit Form.

  4. You can view every success API of the submitted data from the timeline of related info.
