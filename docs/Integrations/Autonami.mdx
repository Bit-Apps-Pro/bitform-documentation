Another integration that eases your email marketing is **Autonami integration** via Bit form. When users submit any form, the email of that corresponding user will store in the newsletter list of Autonami. For more details go through this documentation carefully.

## Authorization Settings :

1. At first, select the form name you want to apply the Autonami integration; after that, follow the below instructions:<br/>

  **Settings=> Integrations => Click on (+) => Click on ‘Autonami.’**

  <Image src="Autonami/Create Form" alt="Autonami" w="1020" h="400" />
  <Image src="Autonami/Setting to Integration" alt="Autonami" w="1020" h="400" />
  <Image src="Autonami/Bit Form Integration" alt="Autonami" w="1020" h="400" />

2.	Before setting up the integration, you must confirm that the **Autonami Pro plugin** is activated on your site. Otherwise, the integration will not work.

3. Integration Name: Set the integration name **“Autonami API”** And click on the  **Authorize** button to complete authorization settings. Then Click on next button.

  <Image src="Autonami/Authorization" alt="Autonami" w="1020" h="400" />

## Integration Settings : 

1.	After completing the authorization process, select the contacts list that you have created before on Autonami. You can choose multiple lists from here. You can add a Tag [ also multiple tags] for the contacts that you have created before on your Autonami account. Your Autonami plugin must already have tags.

  <Image src="Autonami/List Select" alt="Autonami" w="1020" h="400" />
  <Image src="Autonami/Tag Added" alt="Autonami" w="1020" h="400" />

2.	Then you have to map the form fields with Autonami fields. By default, the field is Email. Also, you set any field required on Autonami, and it will be automatically displayed on your map settings. You can set multiple fields from map field settings.

  <Image src="Autonami/Map Fields" alt="Autonami" w="1020" h="400" />

## Actions

1.	If the contact already exists in Autonami. It will be skip contact; otherwise, it will update the contact based on email address.

  <Image src="Autonami/Action" alt="Autonami" w="1020" h="400" />

2. After clicking **Finish & Save** you can see the integration you have created now.

  <Image src="Autonami/Finish & Save" alt="Autonami" w="1020" h="400" />
  <Image src="Autonami/Completed List" alt="Autonami" w="1020" h="400" />

3. After completing the settings, you have to set the conditions when this integration works. Don’t forget to specify the integration name; otherwise, the integration will not work correctly. Here we have set the condition that – every time when records will be created/ edited, on form submission the integration will be triggered. And update the form by clicking the **“Update”** button.<br/>
  <Image src="Autonami/conditional Logic" alt="Autonami" w="1020" h="400" />

4. Now, whenever a user submits the form, it will be added to the corresponding contacts list that you have selected in integrations. Also can view from form responses of Bit Form.

  <Image src="Autonami/Contact List" alt="Autonami" w="1020" h="400" />
  <Image src="Autonami/Bit Form Responses" alt="Autonami" w="1020" h="400" />

5.	You can view every success API of the submitted data from the timeline of related info.

  <Image src="Autonami/Related Info" alt="Autonami" w="1020" h="400" />
