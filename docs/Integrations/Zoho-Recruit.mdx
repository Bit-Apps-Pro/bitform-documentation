Using **Zoho Recruit integration** with WordPress form builder, you can automatically push your WordPress form data in Recruit modules. You can easily integrate without any coding. This Documentation goes through the configuration process.

First, you have to log-in to your WordPress dashboard. Then click on Bit form from the left sidebar. Select an existing form that you want to integrate with Zoho Recruit if you don’t have any form please create a new form. Please follow the screenshots below:

<Image src="Zoho Recruit/1.create_form" alt="Zoho Recruit" w="1020" h="300" />

Then click on the Settings menu from the top navigation bar.Navigate to ***Settings->Integrations->Click on (+) ->Select Zoho Recruit**. Please follow the screenshots below:

<Image src="Zoho Recruit/2.Homepage" alt="Zoho Recruit" w="1020" h="300" />

<Image src="Zoho Recruit/3.Bit-Form-to-Zoho-Recruit-integraton-select" alt="Zoho Recruit" w="1020" h="300" />

Select a data center from the dropdown. Then click on the ***Zoho API Console***. Navigate to ***API Console->Get Started->Server-based Applications***
. Please follow the screenshots below

<Image src="Zoho Recruit/4.Recruit-768x364" alt="Zoho Recruit" w="1020" h="300" />

<Image src="Zoho Recruit/5.GetStarted" alt="Zoho Recruit" w="1020" h="300" />

<Image src="Zoho Recruit/6.api-console-server-based" alt="Zoho Recruit" w="1020" h="300" />

Fill those fields (Homepage URL and Authorized Redirect URLs) the same as your Integration Homepage. Then click UPDATE.

<Image src="Zoho Recruit/7.Client-Details" alt="Zoho Recruit" w="1020" h="300" />

A Client ID and a Client Secret will be generate. Then go to Client Secret tab.

<Image src="Zoho Recruit/8.Client-Secret" alt="Zoho Recruit" w="1020" h="300" />

Then copy the Client ID and Client Secret and paste them to the Bit Form integration page. Then click on the Authorize button (wait for a few seconds). After the successful authorization click on the Next button.

<Image src="Zoho Recruit/9.Bit-Form-to-Zoho-Recruit-Authorize" alt="Zoho Recruit" w="1020" h="300" />

<Image src="Zoho Recruit/10.Accept" alt="Zoho Recruit" w="1020" h="300" />

<Image src="Zoho Recruit/11.Bit-Form-to-Zoho-Recruit-Authorization" alt="Zoho Recruit" w="1020" h="300" />

Select the required Recruit module from the dropdown.

<Image src="Zoho Recruit/12.Module-1024x491" alt="Zoho Recruit" w="1020" h="300" />

Map all the mandatory Recruit fields to the corresponding fields in your form.

<Image src="Zoho Recruit/13Field-Mapping" alt="Zoho Recruit" w="1020" h="300" />

If you use File Upload field should to map this separately.

<Image src="Zoho Recruit/14.File-Upload" alt="Zoho Recruit" w="1020" h="250" />

You can send data to Zoho Recruit related list from Bit Form. Select the required related list module and layout from the drop-down provided. Then map the mandatory field. To map more fields, click on the (+).

<Video youtube src="OFHoPhjJoLY" />

<Image src="Zoho Recruit/15.RelatedList" alt="Zoho Recruit" w="1020" h="300" />

**Actions:**

***1.Workflow:*** Select this option to trigger the Zoho Recruit workflow for the selected module. If you didn’t select this option Zoho Recruit workflow for the selected module doesn’t work.

<Video youtube src="dfDmU0Q3sBE" />

<Image src="Zoho Recruit/16.Workflow" alt="Zoho Recruit" w="1020" h="200" />

***2.Approval***: Select this option to automatically send form data to your Zoho Recruit approval list instead of adding them.

<Video youtube src="OyY13BFTLvU" />

<Image src="Zoho Recruit/17.Approval" alt="Zoho Recruit" w="1020" h="150" />

***3. Upsert Record:*** When you choose the ***Upsert Record*** option, if a record with the identical value exists in Zoho Recruit, then it will be updated with the new values. If not, a new record will be created.

<Video youtube src="gexJKzSp0hE" />

<Image src="Zoho Recruit/18.Upsert" alt="Zoho Recruit" w="1020" h="150" />

Select this option you can set a record owner of the sent record. For this, you have to copy the Owner ID from Zoho Recruit and paste it here.

<Video youtube src="gexJKzSp0hE" />

<Video src="19.Bit-Form-to-Zoho-Recruit-integration-owner-id-" alt="Zoho CRM" w="800" h="400" />

Map all the fields, select the necessary Actions to be performed on integration and then click on the Save button to save your settings.

**Supported Field Types  :**

These are the Bit Forms fields that can be mapped with the respective fields in Zoho CRM.

<Image src="Zoho Recruit/Fields1" alt="Zoho Recruit" w="1020" h="400" />

<Image src="Zoho Recruit/Fields2" alt="Zoho Recruit" w="1020" h="350" />

In the end, create a Workflow for triggering the integration. Select the integration which you created before.
