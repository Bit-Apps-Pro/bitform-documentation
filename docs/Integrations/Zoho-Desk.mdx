By the Zoho Desk integration with WordPress form builder, you can automatically create tickets in Zoho Desk. You can easily integrate without any coding. This Documentation goes through the configuration process. 

First, you have to log-in to your WordPress dashboard. Then click on Bit form from the left sidebar. Select an existing form that you want to integrate with Zoho Desk if you donâ€™t have any form please create a new form. Please follow the screenshots below:

<Image src="Zoho Desk/1.create_form" alt="Zoho Desk" w="1020" h="400" />

Then click on the Settings menu from the top navigation bar. Navigate to Settings->Integrations->Click on (+) ->Select Zoho Desk. Please follow the screenshots below:

<Image src="Zoho Desk/2.Homepage" alt="Zoho Desk" w="1020" h="400" />

<Image src="Zoho Desk/3.Bit-Form-to-Zoho-Desk-integration-select" alt="Zoho Desk" w="1020" h="400" />

Select a data center from the dropdown. Then click on the Zoho API Console. Navigate to API Console->Get Started->Server-based Applications. Please follow the screenshots below

<Image src="Zoho Desk/4.Bit-Form-to-Zoho-Desk-api-console" alt="Zoho Desk" w="1020" h="400" />

<Image src="Zoho Desk/5.GetStarted" alt="Zoho Desk" w="1020" h="400" />

<Image src="Zoho Desk/6.api-console-server-based" alt="Zoho Desk" w="1020" h="400" />

Fill those fields (Homepage URL and Authorized Redirect URLs) the same as your Integration Homepage. Then click UPDATE.

<Image src="Zoho Desk/7.Client-Details" alt="Zoho Desk" w="1020" h="400" />

A Client ID and a Client Secret will be generated.Then go to the Client Secret tab.

<Image src="Zoho Desk/8.Bit-Form-to-Zoho-Desk-client-secret" alt="Zoho Desk" w="1020" h="400" />

Then copy the Client ID and Client Secret and paste them to the Bit Form integration page. Then click on the Authorize button (wait for a few seconds). After the successful authorization click on the Next button

<Image src="Zoho Desk/9.Bit-Form-to-Zoho-Desk-integration-homepage" alt="Zoho Desk" w="1020" h="400" />

<Image src="Zoho Desk/10.Bit-Form-to-Zoho-Desk-accept" alt="Zoho Desk" w="1020" h="400" />

<Image src="Zoho Desk/11.Bit-Form-to-Zoho-Desk-authorizations" alt="Zoho Desk" w="1020" h="400" />

Select the required Portal and Department from the dropdown. Map all the mandatory Desk fields to the corresponding fields in your form. At last select the requires Ticket Owner from the actions.

<Video src="12.Bit-Form-to-Zoho-desk-map-field-1024x497" alt="Zoho Desk" w="1020" h="400" />

To map more form fields to Desk fields, click on the (+). The form fields that are relevant to the selected field would be listed in the drop-down under Bit Forms.

<Video src="13" alt="Zoho Desk" w="800" h="400" />

Actions :

1.Ticket Owner: Ticket Owner is mandatory in Zoho Desk. You have to select the Ticket Owner from actions

<Video src="14.Bit-Form-to-Zoho-desk-ticke-owner-1024x497" alt="Zoho Desk" w="1020" h="400" />

2. Product Name: You can product-wise assign ticket in Zoho Desk. For this, you have to select the Product Name from the actions.

<Video youtube src="FrEcyDi3VOc" />


<Video src="15.Bit-Form-to-Zoho-desk-product-name-1024x497" alt="Zoho Desk" w="1020" h="400" />

3. Attachment: To send an attachment to the Zoho Desk you have to select the required file upload field from actions.

<Video src="16.Bit-Form-to-Zoho-desk-attachment-1024x497" alt="Zoho Desk" w="1020" h="400" />

Select the necessary Actions to be performed on integration and then click on the Save button to save your settings.

In the end, create a Workflow for triggering the integration. Select the integration which you created before.

<Video src="17.Bit-Form-to-Zoho-desk-workflow" alt="Zoho Desk" w="1020" h="400" />
