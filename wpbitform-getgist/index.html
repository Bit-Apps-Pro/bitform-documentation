Bit Form Integration With Getgist

Authorization Process​

  1. At first, select the form you want to apply the Getgist integration. If you don’t have any then create a new one from Create Form. After that, follow the below instructions. Follow the path:

    Settings => Integrations => Click on (+) => Click on ‘Getgist’.

  2. Integration Name: Set the proper integration name. Default it's taken Getgist.

Integration Settings​

  1. After that, you have to copy the API key from your Getgist account & paste the key on the field.
    To get the API key,

  • click on Getgist API Console or go to your Getgist dashboard

  • click on Account Setting => Setting => API Key.

  • Now copy the API key and paste it into the integration’s dashboard. And Click on Authorize button.

  1. Click on Next to set up the other integration settings ahead.

  2. After completing the authorization process, Select the User Type from the drop-down option User or Lead.

  3. You will find the Map Field option. You have to map all the necessary fields with the related Getgist field. By default, Email field of Getgist will be available. You can also send custom values by mapping your required field. then, click on the Next button to move forward for further settings.

  4. After clicking Finish & Save you can see the integration you have created now.

  5. After completing the settings, you have to set the conditions when this integration works. Don’t forget to specify the integration name; otherwise, the integration will not work correctly. Here we have set the condition that – every time when records will be created/ edited, on form submission the integration will be triggered.

  6. Now, whenever a user will submit the form it will be added to the corresponding Campaign list of Getgist. Also can view from form responses of Bit Form.

  7. You can view every success API of the submitted data from the timeline of related info.

Actions​

Add Tags: You can enable the action Add Tags Field to add the tags of the user. In addition, you can add multiple tags from the Drop-Down list. If you don't see any Members. Click on the refresh button.
