How to create a WordPress form using BitForm

Create your first form from scratch‚Äã

If you have successfully installed & activated Bit Form, now it's time to create your very first form using Bit Form

Form builder‚Äã

  1. To create a form from scratch, click on Create Form. Then click on the blank form to create the first form. Also, you can use a pre-built form like Contact Form here. No hassle for creating the same form again & again for your multiple websites.



  2. Now, from the builder option, edit the name of the form as your preference.

  3. From the left panel of the page, you can access toolbar, which contains 19+ form fields

    info

    To know more information about types of field, check the Form Fields documentation


  4. The right panel contains the options- Field Settings & Style Customization.

  5. Using the drag & drop form fields & the built-in style editor option, you can customize your form regardless it is an easy or advanced level form. To add a form field, you can click on the field or drag the field from the panel and drop it on the place you want.

  6. To edit the fields you have to hover on the fields that you want to customize. You can also edit the field by clicking on the individual field setting option. In the same way, you can set the field settings, you can customize the field properties.

    info

    To know more information about field customization, check the Field customize documentation

  7. If you want to delete any field, simply hover to the specific fields & press the cross button here.

  8. As we have said before, Bit Form is device-friendly; hence, you can see the live for preview of Tablet, Mobile, Desktop version clicking the marked option.

  9. Once you are done with field settings, save the form.

How you can embed short code‚Äã

There are three ways you can display the form you have created with Bit Form on your desired page. The ways are described below-

1. Add Bit Form using Short Code:

After creating a form you can view that from the form list of Bit Form with the generated short code. You have to copy the short code & simply paste it on the page where you want to show the form to the user.

2. Add Bit Form using Gutenberg Block:

Navigate to the page where you want to show the form. Click on to choose a block & then on +(Add block) option. Now you can see the block called Bit Form & click on that block. After that you have to choose the form you created earlier from the block. So you are set to live on that page on your site.

3. Add Bit Form using Elementor page:

Click on the Edit with Elementor of the page where you want to display the form. Navigate to the Elements. Search for the widget Shortcode, drag & drop the widget on the page. Now copy the form shortcode from the form list of Bit Form & paste that on the input field called Enter your shortcode. You can see the preview of your form layout on the right middle side of the page. Update the page to view the page on your website.

Responses‚Äã

  • Once you are done with setting your form on the front end, you can see the submission on the Bit form- Form Responses page.

  • You can edit, duplicate & delete the submissions from the responses page if you need to. You can simply run these actions by clicking the three dots attached at the right corner side of every single response. Also, if you click on entry responses, you can edit & delete the particular entry.

  • You can view related info of the submission from the Response Details option.

  • Related Info contains two option - Timeline & Note. If you edit any entry data, then the details of that specific edited entry will be listed on the Timeline option. Also, you can save a Note if it is necessary for that submission.

Settings‚Äã

1. Form Settings: To navigate to the form settings, click on the Settings option from the top of the form builder page. Settings option contains :

  • Allow single entry for each IP address.

  • Capture GCLID

  • Disable form after limited entry

  • Limit Form Submission period

  • Blocked IP list

  • Allowed IP

2. Confirmation: You can customize different kind of success/error message that will be shown when user submit any form. Also can set Redirect page & Webhooks.

3.Conditional Logic: In Bit Form, conditional logic are some set of actions which are executed when certain specific condition will be triggered. When user submit any form, the following workflow will be triggered & some actions will be held .

info

To know more information, check the Conditional Logic documentation

4.Email Templates: When user will submit any form if you want to notify via email notification. then you can set customized email body from Email Templates.

5.Integration: You can integrate Bit Form with 30+ third party app to collect user information that can help you to track your lead.

We hope this documentation will make it eaiser for you to create your first form
